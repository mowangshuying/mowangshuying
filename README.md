## 其他事项

* 项目格式请以 abc-abc-abc格式整改，fork的项目命名格式为：abc-fork

* 自编码的开源项目请使用 驼峰命名格式，如：

  > 变量：
  >
  > ```
  > std::string helloWorldStr;
  > ```
  >
  > 函数：
  >
  > ```
  > void helloWorld();
  > ```
  >
  > 类：
  >
  > ```
  > class HelloWorld
  > {
  > public:
  > 	void helloWorld();
  > public:
  > 	std::string helloWorldStr;
  > };
  > ```
